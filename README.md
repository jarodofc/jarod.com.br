# Documentação — MeR Seminovos

Sistema reconstruído do zero.

## Acessos
- Administrador: JAROD / 4922
- Visualização: MER / 123456

## Estrutura
Cada veículo possui menu próprio:
- Resumo
- Documentação
- Venda
- Multas e débitos
- Histórico
- XML
- Exportar

## Funções
- Cadastro e edição de veículos
- Filtro por status e busca
- Múltiplas multas com órgão, código e valor
- Soma automática de débitos
- Controle de documentos
- Vendas mensais
- Relatório CSV
- Impressão/PDF
- Importação XLS/XLSX
- Extração textual de ATPV/CRLV
- Importação XML individual
- Backup completo
- Usuários e permissões
- Configurações da loja
- Histórico por veículo

## Uso
Abra `index.html` em um servidor local ou publique a pasta em Netlify, Vercel ou GitHub Pages.
